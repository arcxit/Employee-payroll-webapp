# Employee Payroll Web App (Python + Flask)

## 📌 Overview
A **Payroll Management Web Application** built using **Python Flask** with an **SQLite backend**.  
This project demonstrates CRUD operations, REST APIs, and integration of a lightweight database.

## 🚀 Features
- Employee registration and salary management
- Update and delete employee records
- REST API endpoints for CRUD operations
- SQLite backend with SQLAlchemy ORM
- Basic HTML/CSS frontend

## 🛠️ Tech Stack
- Python Flask
- SQLite + SQLAlchemy ORM
- HTML, CSS
